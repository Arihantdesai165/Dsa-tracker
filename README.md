# DSA Tracker Command Center

A premium, full-stack application for tracking Data Structures and Algorithms practice. Built with a sleek Cyberpunk aesthetic, dynamic micro-animations, and glassmorphic UI elements.

## Core Features

- **Dashboard**: Track your overall progress, topic completion, and recent activity with dynamic glassmorphic cards.
- **Custom Questions**: Add your own DSA problems via a sleek modal. Complete with auto-validations and instant UI cache syncing.
- **Sorting & Filtering**: Dynamically sort questions by difficulty, name, and recency natively across the backend.
- **Spaced Repetition**: A robust revision scheduling system powered by a PostgreSQL/Drizzle backend to remind you when to review older questions.
- **Topics Tracking**: Organize algorithms by topic and keep tabs on how much you've mastered.
- **Premium Aesthetics**: Fully responsive dark mode with neon indigo/purple accents, floating hovers, and animated loading states.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, shadcn/ui, TanStack Query, Wouter
- **Backend**: Node.js, Express, PostgreSQL, Drizzle ORM
- **Deployment**: Configured to run seamlessly via monorepo architecture.

## Getting Started

To initialize the project locally:

```bash
# Install dependencies
pnpm install

# Start the full stack application
pnpm run dev
```
